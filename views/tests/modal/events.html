Modal Test: Events

Related JIRA Ticket. This test demonstrates how lifecycle events work on the Modal component. Lifecycle events include beforeopen, open, afteropen, beforeclose, close, and afterclose. The two modal triggers below display the order and timing in which these events fire.

The "Show Modal" trigger button creates a modal that has its opening cancelled by using return false; from inside its beforeopen event. Doing this causes rendering of the modal to stop, so it won't show.
