Tabs Test: Resizable Tab Containers

A bug was reported where the Tabs Control would not re-evaluate its number of overflowed tabs when an element containing the Tab Container was resized. This test attempts to demonstrate that this functionality works as intended.

Use the slider control below to adjust the width of the Tab Container. If it's working properly, the spillover menu should always show the correct number of invisible tabs.
