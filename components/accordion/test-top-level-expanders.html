Accordion Test: Left-aligned Expanders on Top-Level Headers

When invoking the accordion, it's possible to change the "displayChevrons" setting to false, which will give headers with expandable panes a left-aligned (+/-) button instead of a Chevron to indicate that it can be expanded.

This visual test makes sure that all possible header combinations work well with left-aligned expanders.
